# 🛍️ Online Retail Sales Analysis

Анализ продаж британского интернет-магазина сувениров за 2010–2011 гг.  
Инструменты: **SQL (SQLite + Beekeeper Studio)** · **Excel (визуализация)**

---

## 📦 Датасет

- **Источник:** [Online Retail Dataset — Kaggle](https://www.kaggle.com/datasets/ulrikthygepedersen/online-retail-dataset)
- **Содержание:** транзакции интернет-магазина: товары, количество, цена, страна, дата заказа

---

## 🎯 Цели проекта

- Рассчитать выручку (Quantity × UnitPrice) и провести её аналитические срезы
- Выявить ключевые рынки сбыта по географии
- Определить сезонные тренды и пиковые периоды
- Найти наиболее маржинальные товарные позиции

---

## 🔍 SQL-запросы

| Анализ | Метод |
|---|---|
| Структура таблицы | `PRAGMA table_info()` |
| Выручка по странам | `GROUP BY Country` + `SUM(Quantity * UnitPrice)` |
| Топ-10 товаров по продажам | `GROUP BY Description` + `ORDER BY SUM(Quantity) DESC` |
| Динамика выручки по месяцам | `strftime('%Y-%m', InvoiceDate)` + `GROUP BY` |

---

## 📊 Ключевые выводы

### 🌍 География
- **Великобритания** — доминирующий рынок как по объёму заказов, так и по совокупной выручке

### 📅 Сезонность
- В летний период (июнь–август) фиксируется устойчивый нисходящий тренд выручки — типичная сезонная просадка спроса на сувенирную продукцию
- Начиная с **сентября 2011** наблюдается резкое восстановление: выручка вырастает с 682 823 до 1 019 898 £
- **Ноябрь 2011** — пиковый месяц по выручке, что характерно для предновогоднего потребительского спроса

### 🏆 Топ-товар
- Лидер продаж — **SET OF 2 TEA TOWELS I LOVE LONDON** (кухонные полотенца с лондонской тематикой)
- Позиция демонстрирует значительный отрыв от ближайших конкурентов: разрыв по выручке составляет порядка **5 000–6 000 £**, что свидетельствует о высокой концентрации спроса на данный артикул
